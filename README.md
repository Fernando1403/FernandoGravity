# FernandoGravity

FernandoGravity is a premium task management and productivity application inspired by the design philosophy of the "Sundays" app. Built for high performance and visual excellence, it features a sleek navy blue theme with a modern glassmorphism UI.

## 🚀 Features

### 📊 Dashboard
- **Recommended Categories**: Quick access to common task types.
- **Active Projects**: Track your ongoing work with visual progress bars and countdowns.
- **Interactive UI**: Clean, responsive layout designed to wow.

### 📋 Task Management (Minhas Tarefas)
- **Dual View System**: Switch seamlessly between **List** and **Kanban** visualizations.
- **Drag-and-Drop**: Fully integrated HTML5 drag-and-drop for both List and Kanban views.
- **List View**:
    - Tasks grouped by status (A FAZER, Em Progresso, Em Revisão).
    - Status badges, priority levels, and progress tracking.
    - Inline task creation within specific status sections.
- **Kanban Board**:
    - Flexible columns: Add, rename, or delete columns dynamically.
    - Interactive cards with grip handles for easy movement.

### 🛠️ Functionality
- **Task CRUD**: Create, view, and delete tasks/projects via an animated slide-up modal.
- **Dynamic Context**: Task creation intelligently pre-selects the category/column based on where you clicked.

## 🎨 Design Philosophy
- **Navy Blue Palette**: Professional and deep `#001f3f` theme.
- **Glassmorphism**: Subtle blurs and borders for a premium feel.
- **Outfit Typography**: Modern, clean, and highly readable.
- **Micro-animations**: Smooth transitions and hover effects for a delightful UX.

## 💻 Tech Stack
- **React 18**
- **Vite** (Build tool)
- **Lucide React** (Iconography)
- **React Router DOM** (Navigation)
- **CSS3** (Custom design system with CSS Variables)

## 🏁 Getting Started

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/gonzales-alexandre/fernando-gravity.git
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run the development server**:
    ```bash
    npm run dev
    ```
4.  **Open the app** at `http://localhost:5173`.
